## введедения
первая версия нашего робота была сделана с помощью EV3 конструкциию, однако столкнувшись проблемами,нами было принято решения пересобрать робот.
## наше решения
робот был собран из деталей Lego EV3.были использованы следущие моторы и дтчики:большой мотор,ультрафиолетовый датчик 2 штук,цветовой датчик 2 штук.
программа для робота была написана с помощью LEGO MINDSTORMS. поворотная система была реализована датчика,которые измеряют расстояния внутренних и внешних стен,при отсутсвии внешних или внутренних стен,делается поворот.Поворотная система была реализована с помощю набора EV3,наш робот заднеприводный,расположенный спереди средний мотор поворачивает,а задние колеса начинают движения,мы решили что данный тип привода робота будет наиболее оптимальным вариантом
